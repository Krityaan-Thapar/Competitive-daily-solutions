# Question 

_Difficulty: Easy_

Print the following pattern
Pattern for N = 4
   *
  ***
 *****
*******

**Examples :**
```
Sample Input 1 :
2
1
2   

Sample Output 1 :
   *
   * 
  ***

Sample Input 2 :
2
3
4

Sample Output 2 :

    *
   ***
  *****
    *
   ***
  *****
 *******
```

**Constraints:**
```
1 <= T <= 100
1 <= N <= 100
```
